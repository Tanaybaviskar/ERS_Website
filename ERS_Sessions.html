OpenCV

OpenCV is an open-source library designed for computer vision and machine learning applications. OpenCV plays a vital role in enhancing the capabilities of robots, enabling them to see, understand, and interact with their environment effectively.

YBF Robot

A yellow ball-following robot is an autonomous system real-time computer vision and control systems to track and follow a specific object of interest. By combining real-time vision algorithms with precise motor actuation, it demonstrates fundamental concepts in robotics, control theory, and computer vision.

ROS

ROS (Robot Operating System) is an open-source framework that provides the tools and libraries necessary for building robotic software, facilitating modular design through nodes that communicate over a network. This modularity allows developers to reuse components and collaborate effectively.

View Event Report

Microcontroller

A microcontroller is a compact integrated circuit designed to perform specific tasks like controlling devices or processing inputs. They consist of a processor, memory, and input/output (I/O) peripherals on a single chip.

ROS2

ROS 2 (Robot Operating System 2) is an open-source framework designed to develop, deploy, and operate robotic applications. It is the next generation of the original ROS framework, built to address modern robotic challenges, including real-time performance and reliable communication.
